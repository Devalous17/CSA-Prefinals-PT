This repository contains my performance tasks for Systems Architecture and Operating Systems, focusing on distributed storage, hardware redundancy, and process scheduling.

## 📂 Distributed Storage (GlusterFS)
Implementation of a 2-node distributed storage cluster using **GlusterFS**. This project demonstrates High Availability (HA) where data remains accessible even if one node fails.
*   **Demo Video:** [Watch GlusterFS Failover Demonstration](https://drive.google.com/file/d/1U34Cri-TXm78QHdSSfE9dpNwofP0Jn1q/view?usp=sharing)
*   **Key Concept:** Real-time data replication and self-healing storage.

## 💾 Hardware Redundancy (RAID 1)
Setup and testing of a **RAID 1 (Mirroring)** array using `mdadm` on Ubuntu Server.
*   **Demo Video:** [Watch RAID 1 Persistence Test](https://drive.google.com/file/d/1rDcnvpCgui9_NKbqKj_mz5J98x1-IEVe/view?usp=sharing)
*   **Key Concept:** Proving data integrity by simulating a physical disk failure while maintaining system uptime.

## ⚙️ Operating System Algorithms
Python implementation of various Disk Scheduling algorithms used by operating systems to manage I/O requests.
*   **Algorithms Included:** FCFS (First-Come, First-Served), SSTF (Shortest Seek Time First), and SCAN.
*   **Goal:** Minimizing total head movement to optimize disk performance.
